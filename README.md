# Hostel Management Portal

A web-based Hostel Management System developed using PHP, MySQL, and XAMPP.

## Features
- Student Registration & Login
- Admin Approval System
- Room Allocation
- Student Dashboard
- Secure Authentication

## Technologies Used
- PHP
- MySQL
- HTML, CSS
- XAMPP
- Git & GitHub

## Setup Instructions
1. Clone the repository
2. Import `hostel_db.sql` into phpMyAdmin
3. Configure `config/database.php`
4. Run on `localhost`
